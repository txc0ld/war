# Polyhaven Model Credits

All models in this directory are sourced from [Polyhaven](https://polyhaven.com/models)
and are released under the [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) license — free for any use, commercial or otherwise, no attribution required.

Each subfolder contains a glTF (.gltf), its binary buffer (.bin), and a `textures/` folder with diffuse + normal + AO/roughness/metallic (ARM) maps at 1k resolution.
